## 数学思维
生活可能会欺骗你,但数学不会, 因为不会就是不会. 数学思维在生活中有很多用法, 譬如

### 把大问题拆成变量

昨天我很焦虑,但我又不知道我为什么焦虑,我就把焦虑这个问题拆开看是睡眠不足, 工作压力, 还是关系紧张. 然后我发现是因为某一个合作团队焦虑,我就马上给他打电话把他说了一顿. 把一个模糊的感受拆成几个可观察的因素, 你的控制感会明显上升.

### 先定义问题, 再表达情绪

遇到问题不要慌, 先问一句, 这个问题本质是什么
是时间不够, 还是优先级没排好
是赚的不够, 还是成本控制有问题
然后拿支笔出来把问题写成一句清晰的话, 问题就已经解决一半了

### 用概率思维替代幻想

很多韭菜式焦虑, 本质上是把小概率当成必然
比如看到某个币一天涨了30%, 就开始幻想自己如果重仓早就翻倍
看到群里有人晒单, 就默认别人天天赚钱
自己亏一次, 就觉得市场专门针对自己
这些都是典型的概率错觉

### 做决策时列条件

任何选择都可以写下三件事, 成本是什么, 收益是什么, 最坏情况是什么. 当最坏情况在可承受范围内, 决策会更果断.

### 用复利思维对待能力

每天减肥0.1%, 看起来很小,一年后会发现完全坚持不下去. 阅读, 运动, 学习, 都适合用复利看待.

久而久之你就会发现学了数学遇到问题大脑自动开始建模. 所以如果孩子问学数学有什么用, 可以坦白说, 很多公式将来不会直接出现在生活里. 但推导时养成的专注力, 拆解问题的习惯, 会陪他很多年.
